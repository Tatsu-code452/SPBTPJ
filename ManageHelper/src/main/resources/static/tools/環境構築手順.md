# NODE.js 環境構築手順

## NODE.JS
### バージョン確認
```
node -v
npm -v
```

### 初期化
```
npm init -y
```

### 実行
```
node .\server\nodeServer.js
```

### install
```
npm install ws
npm install cookie
npm install express
```

## Electron
### 初期化
```
npm install electron
```

### 実行
```
npm start
```
